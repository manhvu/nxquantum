## Scope Summary

v0.7 focuses on:

1. external-operations interoperability contracts (machine-readable, versioned),
2. observability schema/profile governance,
3. standalone production + integration profile enablement without orchestration scope creep.

## Required Verification Outcomes

For v0.7 execution:

1. Envelope versioning and serialization determinism checks are assertion-driven.
2. Observability profile checks include mandatory schema keys and redaction invariants.
3. Standalone lifecycle execution remains valid without external orchestration dependencies.
4. External integration profile examples are covered by deterministic feature evidence.

## Unit/Property Linkage Guidance

Target supporting checks under:

1. `test/nx_quantum/providers_capabilities_test.exs`
2. `test/nx_quantum/provider_bridge_test.exs`
3. `test/nx_quantum/provider_adapters_test.exs`
4. `test/nx_quantum/observability_test.exs`
5. `test/nx_quantum/provider_contract_serialization_test.exs`
6. `test/nx_quantum/release_evidence_contract_test.exs`

Minimum invariants:

1. Stable envelope field presence and typed failure mapping.
2. Profile-specific telemetry schema and cardinality constraints.
3. Deterministic redaction for sensitive payload/config fields.
4. No behavior drift in functional lifecycle contracts when observability adapters change.

## Milestone Execution Checkpoints

### Milestone O (P0)

1. F1 coverage is mapped and executable through capability/lifecycle features.
2. Envelope contract changes are versioned and documented.

### Milestone P (P1)

1. F2 schema governance checks are executable across all observability profiles.
2. Redaction and cardinality behavior remain deterministic and enforced.

### Milestone Q (P2)

1. F3 standalone + integration profile docs map to executable feature evidence.
2. Boundary non-goals remain explicit and unchanged in docs/specs.

## Non-Goals (v0.7 Mapping)

1. Centralized scheduling/retry orchestration behavior in NxQuantum.
2. Dashboard/UI/control-plane feature implementation.
3. Unscheduled migration-assurance API rollout from ADR 0007.
