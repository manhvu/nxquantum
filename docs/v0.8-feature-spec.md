# NxQuantum v0.8 Feature Specification

## Release Theme

Live provider execution parity, production-credible compiler value, and troubleshooting depth.

## Scope (R/S/T)

1. **R**: Top-3 provider live execution behind stable typed lifecycle contracts.
2. **S**: Compiler/transpilation value profiles with deterministic diagnostics.
3. **T**: Observability troubleshooting contract expansion with safe custom metadata.

## Contracts

1. Fixture and live lanes preserve the same lifecycle envelope keys:
   - `schema_version`
   - `request_id`
   - `correlation_id`
   - `idempotency_key`
2. Compiler profile selection stays explicit and machine-readable.
3. Observability policies remain deterministic and redaction-safe across profiles.

## Non-goals

1. Provider parity theater with every Python-stack optimization.
2. Control-plane orchestration responsibilities in NxQuantum.
